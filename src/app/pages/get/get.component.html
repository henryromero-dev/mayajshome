GET (Query)

Purpose: The GET module is your tool for precisely retrieving data from your MayaJS backend. It allows you to perform flexible queries based on specific criteria, returning only the records that match your conditions.
Why It's Important: In most applications, you rarely need to fetch all the data at once. The GET module empowers you to efficiently pinpoint the exact information you require, optimizing data retrieval and processing.

Key Components:

  • Schema: The top-level object defining the data model you're working with.
  • Model Names (e.g., "customer"): Properties within the schema object correspond to the names of your models.
  • Filter Criteria: Within each model's object, you specify the filter criteria using attribute-value pairs. The filter criteria structure is the same as used in the Select and Get modules, allowing you to leverage a wide range of operators for precise filtering.
